# MeowVPN API Documentation

MeowVPN is an **API-first** VPN commerce platform. The admin dashboard, Telegram/Bale bots, and subscription portal all consume the same Laravel JSON API.

## Base URL

| Environment | Base |
|-------------|------|
| Local Docker | `http://localhost:8080/api/v1` |
| Production | `https://api.example.com/api/v1` |

## Authentication

### Session (cookie) — dashboard

1. `GET /sanctum/csrf-cookie` (credentials: include)
2. `POST /api/v1/auth/login` with `{ "log": "username", "pwd": "password", "remember": true }`
3. Subsequent requests send session cookie + `X-XSRF-TOKEN`

### Bearer token — automation

`POST /api/v1/auth/token` → `{ "token": "...", "token_type": "Bearer" }`

Use header: `Authorization: Bearer <token>`

### Internal bot service

Header: `X-SVP-Bot-Service-Secret: <SVP_BOT_SERVICE_SECRET>`

## OpenAPI

Interactive OpenAPI is generated by [Scramble](https://scramble.dedoc.co/) at:

- `/docs/api` (HTML)
- `/docs/api.json` (OpenAPI JSON)
